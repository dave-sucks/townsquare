DATABASE (Prisma) — ADD THESE TABLES:

- id (uuid)
- userId (fk)
- name (string, required)
- description (string, optional)
- visibility enum: PRIVATE | PUBLIC (default PRIVATE)
- createdAt, updatedAt

B) ListPlace (join table)
- id (uuid)
- listId (fk)
- placeId (fk)
- addedAt
- note (optional)
- sortOrder (optional int)
- unique(listId, placeId)

Also ensure:
- user can only mutate their own lists
- but public lists can be viewed by anyone (auth optional depending on existing setup)

## Step 3: The Key Query

This ONE query gets everything (places + reviews + tags):

```sql
SELECT 
  p.id,
  p.name,
  p.neighborhood,
  p.lat,
  p.lng,
  lp."sortOrder",
  
  -- Review count from followed users
  COUNT(DISTINCT r.id) as review_count,
  
  -- Reviewer avatars
  JSON_AGG(
    DISTINCT jsonb_build_object(
      'userId', u.id,
      'username', u.username,
      'firstName', u."firstName",
      'instagramHandle', u."instagramHandle",
      'profileImageUrl', u."profileImageUrl",
      'isVerified', u."isVerified"
    )
  ) FILTER (WHERE r.id IS NOT NULL) as reviewers,
  
  -- Tags with categories
  JSON_AGG(
    DISTINCT jsonb_build_object(
      'slug', t.slug,
      'displayName', t."displayName",
      'categorySlug', tc.slug,
      'categoryName', tc."displayName",
      'icon', t."iconName"
    )
  ) FILTER (WHERE t.id IS NOT NULL) as tags

FROM list_places lp
JOIN places p ON lp."placeId" = p.id
LEFT JOIN reviews r ON p.id = r."placeId"
  AND r."userId" IN (
    SELECT "followingId" 
    FROM follows 
    WHERE "followerId" = 'user-dave'
  )
LEFT JOIN users u ON r."userId" = u.id
LEFT JOIN place_tags pt ON p.id = pt."placeId"
LEFT JOIN tags t ON pt."tagId" = t.id
LEFT JOIN tag_categories tc ON t."categoryId" = tc.id

WHERE lp."listId" = 'list-dave-burgers'
GROUP BY p.id, p.name, p.neighborhood, p.lat, p.lng, lp."sortOrder"
ORDER BY lp."sortOrder";
```

## Step 4: Get Reviews for a Place

When someone clicks a pin, get the reviews:

```sql
SELECT 
  r.*,
  u.username,
  u."firstName",
  u."lastName",
  u."instagramHandle",
  u."isVerified",
  u."profileImageUrl"
FROM reviews r
JOIN users u ON r."userId" = u.id
WHERE r."placeId" = $1  -- the place they clicked
  AND r."userId" IN (
    SELECT "followingId" 
    FROM follows 
    WHERE "followerId" = 'user-dave'
  )
ORDER BY r."createdAt" DESC;
```

## Step 5: Filter by Tags

When user selects tags, filter the places:

```sql
-- Find places with ALL selected tags (AND logic)
SELECT p.id, p.name
FROM places p
JOIN place_tags pt ON p.id = pt."placeId"
JOIN tags t ON pt."tagId" = t.id
WHERE t.slug IN ('affordable', 'quick-bite', 'burger')
  AND p.id IN (
    SELECT "placeId" 
    FROM list_places 
    WHERE "listId" = 'list-dave-burgers'
  )
GROUP BY p.id, p.name
HAVING COUNT(DISTINCT t.id) = 3;  -- Must have all 3 tags
```
